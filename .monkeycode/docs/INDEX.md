# AI 内容创作系统文档索引

## 项目概述

AI 内容创作系统是一个智能化的内容生成和管理平台，通过 AI 技术帮助用户基于热点话题快速生成高质量的内容，并支持一键发布到各大平台（今日头条、微博、微信公众号等），实现个人 IP 的快速构建和稳定收入来源的闭环。

### 核心价值

- **批量生成素材**：基于 AI 技术，快速生成多种类型的内容（长文章、微头条、视频脚本、音频脚本）
- **智能热点监控**：实时抓取各大平台的热门话题，为内容创作提供素材来源
- **一键多平台发布**：支持今日头条、微博、微信公众号等多个平台的自动发布
- **数据分析与优化**：追踪内容表现数据，优化创作策略
- **质量评估系统**：AI 自动评估内容质量，提供优化建议

### 技术栈

- **前端**：React 18 + Vite + React Router + TanStack Query + shadcn/ui
- **后端**：Express.js + Node.js
- **数据库**：MongoDB + Mongoose
- **缓存**：Redis + node-cache
- **AI 服务**：OpenAI、百度 AI、讯飞 AI（多模型支持）
- **部署**：Docker + Docker Compose + PM2 + Nginx
