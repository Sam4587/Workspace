## 📖 项目简介

Publisher Tools 是一个支持多平台内容发布的自动化系统，从 TrendRadar 项目分离而来，提供完整的**内容创作→发布→数据分析**闭环能力。

### 核心能力

- 🚀 **多平台发布** - 支持抖音、今日头条、小红书三大平台
- 🤖 **AI驱动** - 集成多种AI提供商，智能内容生成
- 📊 **数据分析** - 完整的数据采集和报告生成
- 🎬 **视频转录** - 视频内容AI转录，一键改写发布
