| 参考资料 | 新增术语或资源时 |

### 5.2 版本控制

#### 版本号规范
```
主版本号.次版本号.修订号
```

- **主版本号**: 重大架构变更或文档重构
- **次版本号**: 新增章节或重要更新
- **修订号**: Bug 修复或小改进

#### 变更记录
每个文档底部应包含变更记录：
```markdown
## 变更记录

| 日期 | 版本 | 变更内容 | 作者 |
|------|------|----------|------|
| 2026-02-19 | 1.0 | 初始版本 | AI助手 |
```
